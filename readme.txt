WPDash Notes

== Description ==

WPDash Notes est un plugin qui vous permet de créer des notes sur votre tableau de bord WordPress et sur ceux des autres utilisateurs.

WPDash Notes vous offre la possibilité d’ajouter des Notes dans votre tableau de bord WordPress. Vous pouvez spécifier la cible (rôles utilisateur et/ou les utilisateurs eux-mêmes), choisir la couleur, autoriser ou pas les commentaires, notifier les destinataires par email, ajouter du texte, des listes, des images, etc…
Vous travaillez avec plusieurs personnes sur un site WordPress ? Vous voulez prendre des notes ? Vous avez besoin d’un pense-bête ? De donner des infos ou des directives ? C’est exactement ce que vous pourrez faire avec WPDash Notes !

= Français =

Fonctionnalités de WPDash Notes :
Le plugin WPDash Notes a d’abord été pensé pour être simple et 100% compatible avec l’environnement du tableau de bord WordPress.
* Notes avec couleurs au choix (fond et texte)
* Notes publiques ou privés ou par type de users
* Ajoutez autant de notes que vous le souhaitez
* Notifications par email des notes nominatives ou par catégorie d’utilisateur
* Commentaires autorisés ou non sur les notes

Ce plugin vous est gentiment proposé par <a href="https://wpformation.com/">WPFormation</a>.

Aucune de nos extensions n’exploite les données des sites où elles sont installées.

= English =

WPDash Notes is a plugin that allows you to create notes on your WordPress dashboard and those of other users.

WPDash Notes adds the functionality to add sticky notes in the dashboard. You can specify the target (user roles and / or users themselves) with notes.
Do you work with multiple people on a WordPress site? Want to take notes? Give info or directions? This is exactly what you can do with WPDash Notes!

Features of WPDash Notes:
The WPDash Notes plugin was originally designed to be simple and 100% compatible with the WordPress dashboard environment.
* Notes with choice of colors (background and text)
* Public or private notes or by type of user
* Add as many notes as you want
* Email notifications of personal notes or by user category
* Comments allowed or not on the notes

This plugin is kindly proposed by <a href="https://wpformation.com/" target="_blank">WPFormation</a>.

None of our extensions exploit data from the sites where they are installed.

== Installation ==

= Français =
1. Aller dans Extensions › Ajouter.
2. Rechercher *WPDash Notes*.
3. Recherchez ce plugin, téléchargez-le et activez-le.

= English =
1. Go to Plugins › Add New.
2. Search for *WPDash Notes*.
3. Look for this plugin, download and activate it.
